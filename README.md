# Sports Equipment Borrowing System

Student ID: 67702499

## Project Description

This project is a web-based sports equipment borrowing system that allows users to manage sports equipment, borrow items, and track borrowing records.

The system is built using PHP MVC architecture and runs in Docker environment.

---

## Features

- User Login
- Equipment Management
- Borrow Equipment
- Borrowing Records
- Dashboard
- Calendar View

---

## Technology Stack

- PHP
- MySQL
- Docker
- MVC Architecture
- JavaScript
- CSS
